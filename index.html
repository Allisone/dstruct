Data Structures & Algorithims for TypeScript and JavaScript

Meeting the demands of modern web applications.

Problem

In JavaScript & TypeScript the only data structures provided are arrays, and string key hash maps. This limits the ability of developers to solve problems; where other data structures would work better with less code and more efficiency. Existing solutions are limited in scope, do not provide static typings for TypeScript, have incompatible APIs, lack modularity, lack separation of interface and implementation, and do not take advantage of the upcoming features of JavaScript.

  • Lack modularity.
  • Lack separation of interface and implementation.
