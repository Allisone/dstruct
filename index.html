Data Structures for TypeScript and JavaScript

Problem

In JavaScript & TypeScript the only data structures provided are arrays, and string key hash maps. This limits the ability of developers to solve problems; where other data structures would work better with less code and more efficiency. Existing solutions have the following pitfalls:

  • Are limited in scope
  • Do not provide static typings for TypeScript
  • Have incompatible APIs
  • Lack modularity
  • Lack separation of interface and implementation
  • Do not take advantage of the upcoming features of JavaScript

Solution

Create a comprehensive library of well-researched data structures which will increase the productivity of JavaScript and TypeScript developers. Release the library as open source with permissive licensing, and accept public contributions as well.

Results

Success was gauged by:

  • Documentation for public APIs
  • Extensive test coverage
  • Published public modules containing the completed library
  • Surveys conducted on various online development communities in order to determine the utility and quality of the library.
